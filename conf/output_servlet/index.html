Competition Network

Visualizing tech companies competition in network diagram format [BETA]

Method 1: Visualization

(This web application is far from perfect and still have plenty of bugs; so feel free to give any feedback)
The visulization program displays network graph of tech companies where they compete with each other. Every node represents one tech company, and each edge indicates that a company is a competition to another company connected with it.
The visualization does not diplay the whole network of tech companies competition; If it does, it would involve 80,000++ nodes. Instead, only subgraph with center node of your choice will be displayed. You can set egocentricity of the graph: it just means how far the furthest node to the center node. Setting the categorized by will color each node according to its distinctive category.
